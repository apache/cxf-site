java.lang.Object
  org.apache.cxf.sts.claims.ClaimsParameters

public class ClaimsParameters
extends Object

This class encapsulates the parameters that will be passed to a ClaimsHandler instance to to retrieve claims. It consists of both parameters that have been extracted from the request, as well as configuration specific to the STS itself.
