Interface BindingFactoryManager

  • All Known Implementing Classes:
    BindingFactoryManagerImpl

    public interface BindingFactoryManager
    The manager interface represents a repository for accessing BindingFactorys. Provides methods necessary for registering, unregistering or retrieving of BindingFactorys.
